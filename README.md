# ShopWave

## Project Description
ShopWave is a full-stack e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript. The platform enables users to browse products, manage shopping carts, and place orders through a responsive web interface.

## Features
- User Registration and Login
- Product Catalog
- Shopping Cart
- Order Management
- Database Integration
- Responsive Design

## Tech Stack
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Apache (XAMPP)
- phpMyAdminn

## Installation
1. Clone the repository.
2. Copy the project folder to XAMPP's htdocs directory.
3. Start Apache and MySQL.
4. Import `mysqlfile.sql` into MySQL.
5. Open `http://localhost/Myshop`.
